# UsageDock v0.2.8

UsageDock `v0.2.8` is a compact tray UI refinement release.

## Highlights

- Reworked the tray header into a tighter single-row layout so more provider content is visible immediately.
- Moved auto-refresh controls and update checks into the top settings panel instead of keeping them expanded in the footer.
- Fixed the interval dropdown so it opens downward and sits cleanly above the provider list.

## Installation

- Windows installer: `UsageDock_0.2.8_x64-setup.exe`
- Platform: Windows x64
- Install mode: per-user, no administrator privileges required
