# UsageDock v0.2.1

UsageDock `v0.2.1` is a patch release focused on clearer reset visibility in provider cards.

## Highlights

- Added reset timing extraction and display for providers that expose reset windows.
- Wired Codex reset windows through from the usage API.
- Improved reset wording in the UI so cards use cleaner phrasing like `Resets in ...` instead of repeated status text.
- Simplified provider captions by removing redundant `live signal` copy when it does not add value.

## Installation

- Windows installer: `UsageDock_0.2.1_x64-setup.exe`
- Platform: Windows x64
- Install mode: per-user, no administrator privileges required
