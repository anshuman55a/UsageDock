# UsageDock v0.2.9

UsageDock `v0.2.9` is a tray-UX polish release focused on readability and lower visual clutter.

## Highlights

- Tightened provider cards and metric spacing while keeping reset timing readable in stacked views.
- Added a smoother settings-panel close animation and keyboard `R` refresh for focused tray use.
- Cleaned up per-card refresh behavior and fixed long provider error text wrapping inside cards.

## Installation

- Windows installer: `UsageDock_0.2.9_x64-setup.exe`
- Platform: Windows x64
- Install mode: per-user, no administrator privileges required
