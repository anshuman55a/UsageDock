# UsageDock v0.2.4

UsageDock `v0.2.4` moves the current `dev` work onto `main` and publishes it as the next public patch release.

## Highlights

- Ships the hardened Tauri configuration and updater-enabled app path from the main release line.
- Includes the provider fixes and tray behavior improvements that were already validated in recent builds.
- Adds the repo-level open-source basics: MIT license, contribution guide, security policy, changelog, and issue and pull request templates.

## Installation

- Windows installer: `UsageDock_0.2.4_x64-setup.exe`
- Platform: Windows x64
- Install mode: per-user, no administrator privileges required
