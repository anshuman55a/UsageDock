# UsageDock v0.2.7

UsageDock `v0.2.7` is a small product polish release focused on Cursor visibility and tighter app copy.

## Highlights

- Fixed Cursor free-plan reporting so included usage percentages render as actual usage lines instead of falling back to `No usage data`.
- Refined the tray header subtitle to `Local AI coding usage at a glance`.

## Installation

- Windows installer: `UsageDock_0.2.7_x64-setup.exe`
- Platform: Windows x64
- Install mode: per-user, no administrator privileges required
