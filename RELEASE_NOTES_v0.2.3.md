# UsageDock v0.2.3

UsageDock `v0.2.3` is a release correction for the first updater-enabled builds.

## Highlights

- Fixed the release workflow so updater artifact generation includes the required `plugins.updater` configuration.
- Unblocked signed updater packaging for both Windows and Linux release jobs.
- Keeps the in-app auto-update path introduced in `v0.2.2`, but ships it from a tag with the corrected CI configuration.

## Installation

- Windows installer: `UsageDock_0.2.3_x64-setup.exe`
- Platform: Windows x64
- Install mode: per-user, no administrator privileges required
